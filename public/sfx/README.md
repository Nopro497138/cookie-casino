
# Sound Effects Folder

Drop your `.mp3`, `.ogg`, or `.wav` files here.
The site works fine without them — missing files are silently ignored.

## Expected filenames

| File | Plays when |
|------|-----------|
| `win.mp3` | Any win |
| `lose.mp3` | Any loss |
| `click.mp3` | Button click |
| `coin.mp3` | Coin awarded |
| `spin.mp3` | Generic spin |
| `roulette_spin.mp3` | Roulette wheel starts |
| `roulette_land.mp3` | Ball lands |
| `slot_spin.mp3` | Slots start spinning |
| `slot_win.mp3` | Slots win |
| `slot_stop.mp3` | Each reel stops |
| `card_deal.mp3` | Card dealt in Blackjack |
| `card_flip.mp3` | Card revealed |
| `blackjack_win.mp3` | Blackjack win |
| `blackjack_bust.mp3` | Bust |
| `mine_reveal.mp3` | Safe cell revealed |
| `mine_explode.mp3` | Mine hit |
| `mine_cashout.mp3` | Cash out in Mines |
| `plinko_drop.mp3` | Ball dropped |
| `plinko_bounce.mp3` | Ball hits a peg |
| `plinko_land.mp3` | Ball lands in bucket |
| `dice_roll.mp3` | Dice rolling |
| `dice_land.mp3` | Result shown |
| `crash_start.mp3` | Crash round starts |
| `crash_fly.mp3` | Rocket flying (loop) |
| `crash_boom.mp3` | Crash happens |
| `crash_cashout.mp3` | Cashed out |
| `fish_cast.mp3` | Line cast |
| `fish_bite.mp3` | Fish bites |
| `fish_catch.mp3` | Catch revealed |
| `fish_junk.mp3` | Junk caught |
| `keno_draw.mp3` | Number drawn |
| `keno_match.mp3` | Matched number |
| `coin_flip.mp3` | Coin flipping |
| `coin_land.mp3` | Coin lands |
| `wheel_spin.mp3` | Wheel starts |
| `wheel_tick.mp3` | Tick between segments |
| `wheel_land.mp3` | Wheel stops |
| `notification.mp3` | New notification |
| `purchase.mp3` | Item purchased |
| `daily_bonus.mp3` | Daily bonus claimed |
| `chat_message.mp3` | New chat message |
